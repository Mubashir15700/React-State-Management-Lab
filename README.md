🚀 Purpose
The goal of this project was to master modern data-fetching patterns by implementing the same CRUD logic using two different ecosystems:
 - Redux Toolkit Query: Managing orders and global UI state.
 - TanStack Query (React Query): Managing product inventory and server-cache invalidation.
 - Observability Layer: Integrating Prometheus and Grafana to monitor API performance, request metrics, and system health in real time.

🛠️ Tech Stack
 - Backend: NestJS, Prisma ORM, PostgreSQL, Event-Driven Architecture.
 - Frontend: React (Vite), TypeScript, Tailwind CSS.
 - State Management: Redux Toolkit, TanStack Query v5.
 - Monitoring & Observability: Prometheus (metrics collection), Grafana (visualization dashboards).

💡 Key Learnings
 - Handling Path & Query Parameters in dynamic API hooks.
 - Implementing Automated Cache Invalidation (Tags vs. Query Keys).
 - Configuring a Monorepo structure for seamless Full-Stack development.
 - Database transactions and event emitters in NestJS.
 - Instrumenting backend services with Prometheus and visualizing metrics using Grafana dashboards.
